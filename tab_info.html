New Tab Information


This webpage is designed to be set as your new tab page.

How set it as your new tab page

WARNING: This will require downloading external content

Firefox

Download this extension Once downloaded, click the new icon in the toolbar at the top of the screen.
Then you select the Custom URL for the option. For the URL put https://clygro.ml/tab.html into the textbox, the changes will be applied and that is all sorted.
