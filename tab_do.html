What New Tab does


This webpage is designed to be set as your new tab page.

New Tab does

-Shows the time using Javascript
-Allows predefined website shortcuts
-Contain links to external content, which may collect user data.

New Tab doesn't

-Collect user data, such as your location, IP address or browser info
-Show the weather(planned for the future)
-Have an accounts system
-Have a search bar (planned for the future)
