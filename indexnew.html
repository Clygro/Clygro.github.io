18 Argons Minecraft Server

18 Argons (formerly known as ClyMC) started as a minigames server in Mid 2020, now it focuses on Survival and Creative servers, the minigames are still available.

How to connect

To connect to Survival join survival.18argons.xyz on Minecraft Java Edition version 1.19.1, other versions might be supported, however it is recommended to play on 1.19.1.

To connect to Creative join 82.26.167.230:25566 on Minecraft Java Edition version 1.19.1, other versions might be supported, however it is recommended to play on 1.19.1.
